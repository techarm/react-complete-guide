# Learn how React sends HTTP requests

## How to Use

This project is to ues mockoon cli to create mock server.

## 1. Install

```
npm install
```

## 2. Start mock api server

```
npm run mock-start
```

## 3. (Optional) List mock api server info

```
npm run mock-list
```

## 4. Start react application

```
npm start
```

## 5. Stop mock api server

```
npm run mock-stop
```
